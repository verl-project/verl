# 硬件规格参考

## 硬件型号与芯片类型对应

| 硬件型号 | 芯片类型代码 | 备注 |
|----------|-------------|------|
| Atlas 200T A2 Box16 | A2 | 也写作 910b |
| Atlas 900 A2 PODc | A2 | 也写作 910b |
| Atlas 800T A3 | A3 | |

**注意：** A2 和 910b 是同一类型芯片的不同写法：
- A2: 芯片类型代码
- 910b: NPU设备型号，也有910B1、910B2、910B3等

## 输入映射

| 用户输入 | 映射结果 |
|----------|----------|
| Atlas 200T A2 Box16 | A2 |
| Atlas 900 A2 PODc | A2 |
| Atlas 800T A3 | A3 |
| 200T A2 / 200T | A2 |
| 900 A2 POD / 900 A2 | A2 |
| 800T A3 / 800T | A3 |
| A2 | A2 |
| 910b | A2 |
| 910B1 | A2 |
| 910B2 | A2 |
| 910B3 | A2 |
| A3 | A3 |
| 910c | A3 |

## 镜像标签构造规则

```
# vLLM后端
verl-{CANN版本}-{芯片类型}-ubuntu22.04-py3.11-{版本}

# SGLang后端
verl-sglang-{CANN版本}-{芯片类型}-ubuntu22.04-py3.11-{版本}
```

### 芯片类型映射到镜像标签

| 映射结果 | 镜像标签中的写法 |
|---------|--------------|
| A2 | 910b |
| A3 | a3 |

例：
- 用户选择: CANN=8.5.0, 芯片=A2 (或910b) → verl-8.5.0-910b-ubuntu22.04-py3.11-latest
- 用户选择: CANN=8.5.0, 芯片=A3 → verl-8.5.0-a3-ubuntu22.04-py3.11-latest
